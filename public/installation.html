Manual Commands

Ubuntu/Debian:
sudo apt update
sudo apt install nodejs npm ffmpeg vlc
NVIDIA Drivers:
sudo apt install nvidia-driver-525
sudo apt install nvidia-cuda-toolkit
AMD Drivers:
sudo apt install mesa-vulkan-drivers
sudo apt install radeontop
FFmpeg with GPU:
sudo apt install ffmpeg
ffmpeg -hwaccels
