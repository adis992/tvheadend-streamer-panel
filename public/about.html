About TVHeadend Streamer

Advanced GPU-accelerated transcoding platform

Advanced TVHeadend Streaming Panel with GPU Transcoding

Features

GPU Acceleration
Hardware transcoding with NVIDIA NVENC, AMD VCE, and Intel QuickSync
Multiple Profiles
25+ predefined transcoding profiles with custom profile support
UDP Streaming
Real-time UDP multicast streaming with bandwidth monitoring
Real-time Monitoring
Live CPU, RAM, GPU usage and bandwidth statistics
Auto Installation
Automatic detection and installation of GPU drivers and dependencies
Demo Mode
Works without TVHeadend server using demo channels

Documentation

This application provides a comprehensive web interface for managing TVHeadend streaming with advanced GPU-accelerated transcoding capabilities. It features individual channel controls, profile management, real-time monitoring, and automatic system component installation.

Quick Start

git clone https://github.com/adis992/tvheadend-streamer-panel.git
cd tvheadend-streamer-panel
chmod +x install.sh
./install.sh
npm start
